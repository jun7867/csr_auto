# 🚀 Gemini API 기반 회고 자동화 사용 가이드

**서버 필요 없음!** 브라우저에서 바로 Gemini API를 호출합니다.

---

## ✨ 장점

- ✅ **서버 불필요** - Node.js, Ollama 설치 필요 없음
- ✅ **빠른 속도** - Gemini 1.5 Flash는 매우 빠름
- ✅ **무료 크레딧** - 월 1,500회 무료 (일 50회 사용해도 충분)
- ✅ **높은 품질** - Ollama보다 훨씬 좋은 회고 품질
- ✅ **간편함** - 스크립트 복사 & 붙여넣기만 하면 끝

---

## 🎯 사용 방법 (매우 간단)

### 1. 회고 페이지 열기
회고를 작성할 페이지로 이동합니다.

### 2. 브라우저 콘솔 열기
- **Windows/Linux**: `F12` 또는 `Ctrl + Shift + J`
- **Mac**: `Cmd + Option + I`

### 3. 스크립트 실행
`CONSOLE_PASTE_API.js` 파일의 **전체 내용**을 복사해서 콘솔에 붙여넣고 `Enter`

### 4. 작업 입력
팝업창이 뜨면 "오늘 한 일"을 입력하고 확인

### 5. 자동 완료 🎉
스크립트가 자동으로:
- ✅ 버튼 클릭
- ✅ 폼 작성
- ✅ Gemini로 회고 생성
- ✅ 제출 완료

## 🔧 API 키 변경하기

현재 제공된 API 키로도 충분히 사용 가능하지만, 본인의 키를 사용하고 싶다면:

### 1. Gemini API 키 발급받기
https://aistudio.google.com/app/apikey

### 2. 스크립트 수정
`CONSOLE_PASTE_API.js` 파일에서:

```javascript
// 이 부분을 찾아서 수정
const API_KEY = "여기에_본인의_키를_넣으세요";
```

## ⚠️ 문제 해결

### "API 키 오류"가 뜨면?
- API 키가 올바른지 확인
- 인터넷 연결 확인
- Gemini API 무료 할당량 확인 (https://aistudio.google.com)

### "폴백 템플릿 사용" 메시지가 뜨면?
- API 호출에 실패해서 기본 템플릿이 사용됨
- 결과물 품질은 떨어지지만 작동은 함
- 인터넷 연결을 확인하고 다시 시도

### 회고가 생성되지 않으면?
- 브라우저 콘솔(F12)에서 에러 메시지 확인
- 스크립트를 다시 복사 & 붙여넣기
- API 키가 유효한지 확인
